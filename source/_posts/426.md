# 树定义
```java
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;
    public TreeNode(int val) {
        this.val = val;
    }
}
```

# 先序遍历
## 先序遍历递归
```java
public void preorder(TreeNode root){
    System.out.print(root);
    preorder(root.left);
    preorder(root.right);
}
```

## 先序遍历非递归
```java
public void preorder2(TreeNode root){
    LinkedList<TreeNode> list = new LinkedList<>();
    while(root != null || !list.isEmpty()){
        if(root != null){
            System.out.print(root);
            list.push(root);
            root = root.left;
        }

        if(!list.isEmpty()){
            root = list.pop();
            root = root.right();
        }    
    }
    
}
```

# 后序遍历
## 后序遍历递归
```java
public void postorder(TreeNode root){
    postorder(root.left);
    postorder(root.right);
    System.out.print(root);
}
```

## 后序遍历非递归
```java
public void postorder2(TreeNode root){
    LinkedList<TreeNode> list = new LinkedList<>();
    LinkedList<TreeNode> list2 = new LinkedList<>(); 
    while(root != null || !list.isEmpty()) {  
        while (root != null) {  
            list.push(root);  
            list2.push(new Integer(0));  
            root = root.left;  
        }  
        while (!list.isEmpty() && list2.peek().equals(1)) {  
            list2.pop();  
            System.out.print(list.pop());  
        }  
  
        if (!list.isEmpty()) {  
            list2.pop();  
            list2.push(new Integer(1));  
            root = list.peek();  
            root = root.right;  
        }  
    }  
}
```
